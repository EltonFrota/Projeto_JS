//strings

var texto = 'Curso de JavaScript'

declaramos uma variavel com a string var
texto: é o nome da nossa variavel
= : sever para atribuir(passar um valor para a variavel)
'Curso de JavaScript' : é a string q vamos armazenar na nossa
variavel.(Podemos usar tanto as aspas simples ou duplas, desde q 
no fechamento use a mesma escolhida).



//number

Nas variaves do tipo number, não existe diferenciação entre inteiro 
e fracionados no JS, pois todos são do tipo numerico.

//boolean

é verdadeiro ou falso

var teste = true   (verdadeiro)
var teste = false  falso)
